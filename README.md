# Certificados

Plugin de WordPress para gestionar y emitir certificados.

## Estado

Proyecto inicial. La estructura base del plugin ya está creada y lista para desarrollar.

## Instalación local

1. Copia esta carpeta dentro de `wp-content/plugins/certificados`.
2. Activa el plugin desde el panel de administración de WordPress.

## Desarrollo

Archivo principal del plugin:

```text
certificados.php
```
